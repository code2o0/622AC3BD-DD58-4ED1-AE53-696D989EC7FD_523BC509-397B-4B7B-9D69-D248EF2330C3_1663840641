Configuration Guide

This option shows some of the cameras on your device, the currently selected camera will be used as the recording camera.

The image quality preset when the current camera is recording, you can choose the optimal preset according to your device performance, if there is a delay in recording, reduce the image quality preset.

The frame rate limit when recording, the default is 24.

The orientation of the device during recording. If the device is landscaped, choose left or right landscape. If you choose Auto, it will be automatically set according to the device orientation when recording starts. This selection only affects the recorded screen orientation, not the App layout.

Whether to use the device as an RTSP server, and if so, RTSP capable players will be able to connect to the device for live preview.

How many RTSP players are allowed to preview in real time at the same time, this option only takes effect when the RTSP server is turned on.

Whether to save the recorded images to a file in real time while recording.

When it comes to writing files, writing will pause if the file saved with this profile reaches the set size. This option affects the real-time writing and motion detection capture duration.

Whether to perform motion detection processing while recording, that is, to detect whether the screen changes while recording.

The sensitivity of processing motion detection. The higher the value, the more sensitive it is, and the easier it is to trigger when the motion detection takes effect. It is recommended to set it according to the scene during recording.

When motion detection is in effect, the screen will be captured and saved, and when it is 0, no capture will be performed.

Whether to detect the presence of faces while recording.

When there is an RTSP player for real-time preview, the video quality transmitted to the player.

When there is an RTSP player for real-time preview, the video bit rate transmitted to the player.

Whether to send a notification when motion detection or face detection takes effect, if enabled, the system will send a request according to the notification method you configured.
